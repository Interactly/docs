Welcome to the Interactly API docs!.

## Where to find

Yon can get API Keys from Interactly.ai [Dashboard](https://dashboard.interactly.ai/api-keys)

## How to use

You can use API Key as a Header to all our API endpoints.

Add a header as `Authorization` with value as `Bearer <api_key_here>`
