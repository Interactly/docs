## Creating and Launching a campaign:

Navigate to the dashboard, select  **campaigns tab**, and click **New campaign** to begin.

<Frame caption="">
  <img src="/images/campaigns/create-new-campaign.png" />
</Frame>

Provide a name for the campaign and select the **'Create'** button. This will redirect you to the page displayed below.

<Frame caption="">
  <img src="/images/campaigns/camaign-creation-page.png" />
</Frame>

The default selection is the **single model**. Choose an assistant from the **'Select Assistant Number'** dropdown menu.

Provide your email address and press 'Enter' to confirm. We will use this address to notify you the moment the campaign is finished.

The default time zone will be America/New York. You can select your time zone from the drop down menu.

Provide **Date**, **Start Time** and **End Time**  fields.

- **Date**: It is the date on which the campaign runs.
- **Start Time**: At which the campaign will start processing.
- **End Time**: At which the campaign will finish running the calls.

Set a value for **Max Parallel Calls**. It is the maximum number of parallel calls that can be run when the campaign is running. 

- In My **organization → License**,  there will be an org limit set by the interactlya.ai  team. You can set the max parallel calls limit up to the Org limit value.

<Frame caption="">
  <img src="/images/campaigns/org-limit.png" />
</Frame>

Provide **webhook configurations**.

- **Webhook URL** : It is the URL at which the webhooks are received .
- **Webhook Secret**: It is the optional value you can set so that for every webhook interactly will send it as an **“x-interactly-secret”** header for you to understand this webhook os from interactly.
- **Subscribed Webhook Events** : We currently support call status and campaign report events to Subscribe.
      -  **Call Status**:  This contains all the individual calls of the campaign, it will trigger immediately when a call is either completed or failed.

### **Status** :

#### **Phone call status** :

      - The status of the call will be ‘received’ when the campaign is launched.Here status can move to either **ready** when the validation is successful  or **failed** when validation is failed.
      - Then the calls will be moved to queued , which means the campaign time is matched.
      - Then calls will be in ringing state and then processing to end state when the call is finished.

#### **Campaign status**:

      - Campaign will be in **processing** state  until the calls are running.
      - Once all the calls are either completed or failed it will move to the completed state.

#### **Edit Campaign**:

      - After saving the campaign if you want to edit the campaign click on the 3 dots and select “Edit campaign”.
      - Save the campaign after making the changes.

#### **View Campaign Details**:

      - Click on the 3 dots and select the “view details” option to see the campaign details.
      - You can edit the campaign form this page also.  

<Frame caption="">
<img src="/images/campaigns/view-campaign.png" />
</Frame> 


### **Clone Campaign**:

      - You  can duplicate a campaign  by clicking on “clone campaign”.
      - Give a name to the campaign and save it.
      -  All the original campaign fields will be added in the cloned campaign.
      - You can edit the campaign to make changes.

<Frame caption="">
<img src="/images/campaigns/clone.png" />
</Frame>

#### **Delete Campaign**:

      - Click on “delete campaign” , to delete the campaign.
      - The campaign along with the calls will be deleted.
      - Once a campaign is deleted , it can’t be restored.

#### **Pause the campaign**:

      - If you want to pause the campaign while the campaign is running, you can do that by clicking on the “pause campaign” from the three dots.
      - The calls that are already running will be continued and the calls that are in queued state will remain in queued until the campaign is resumed.
	
#### **Stop Campaign**: 

      - A campaign can be aborted by clicking on “stop campaign” .
      - All the calls that are not triggered will be aborted.
      - The campaign status will be changed to “completed” once all the running calls are either “finished” or “failed”.
