## Creating and Launching a campaign:

Navigate to the dashboard, select  **campaigns tab**, and click **New campaign** to begin.


<Frame caption="">
  <img src="/images/campaigns/create-new-campaign.png" />
</Frame>

Provide a name for the campaign and select the **'Create'** button. This will redirect you to the page displayed below.

<Frame caption="">
  <img src="/images/campaigns/multi-page1.png" />
</Frame>

Select the campaign type as  **Multi Model**.

**Note**:  In the Multi Model Campaign, these 2 features will be added extra along with the single model campaign features.

### **Pre-Call Message Notification**:

**Before call**:         	

- To automatically send messages before a scheduled call, enable the pre-call Message Notification feature.
- Select the desired time from the drop-down menu, the default setting is 10 minutes. 
- Custom time can also be set, with a minimum allowed duration of 10 minutes.

**Message Type**:

- By default the type will be SMS.
- Right now, we only support SMS campaigns.
     
 **Message**:

- Input the desired message you want to send before the campaign runs.
- To utilize the dynamic variables available through the campaign assistant, simply choose from the provided list. 
- You can click on the variable to copy it.

<Frame caption="">
  <img src="/images/campaigns/precall-msg.png" />
</Frame>

### **After call Message Notification**:

It is the  notification sent right after a call to share important follow-up details.

#### **Call Answered**:

The recipient successfully answered the call. The follow-up message will now be initiated according to the conversation.

- To utilize this feature, enable the "call answered" toggle.
- Enter the text to be sent if the call was answered.
- You can make use of the dynamic variables in text.

#### **Call Not Answered**:

The call was unanswered. A missed call notification will be sent to the recipient, prompting them to    return the call.

- Enable the “call not answered” toggle.
- Enter the text to be sent if the call goes unanswered.

Example Text:

#### **Call Declined / Busy**:

If the call is declined or the recipient is busy, a message will be sent with an option to  reschedule or attempt the call again later.

- Enable the “call declined/Busy” toggle.
- Enter the text to be sent if the call is declined or the user is busy.

#### **Voicemail Left**:

The user will receive a message notifying them about the voicemail.

- To send a voice mail message enable the outcome type in Assistant Analysis. To do this  
- Go to the clinical assistant that is attached to the campaign.
- Open the **“Analysis”** tab.
- Provide **“Outcome Type”** and **“Out come Field”** in Assistant outcome configuration.
- Whatever message is set in the outcome field that will be sent as a voice mail message.

<Frame caption="">
  <img src="/images/campaigns/analysis-outcome.png" />
</Frame>
        
#### **Call Dropped / Technical Failure**

The call was disconnected due to a technical issue. A message will be sent informing the user and providing next steps.

- Enable  “call dropped/Technical Failure” toggle.
- Enter the message you want to send when the is dropped in the middle or for technical error.
